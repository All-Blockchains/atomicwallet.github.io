What is Stellar (XLM)?

Stellar is a decentralized protocol for financial transactions, based on the XRP open code base with its unique native asset called Lumens (XLM). Stellar can be used to send money anywhere in the world between any currency pairs (using Stellar Anchors). For example, the protocol supports sending money in dollars and receiving in euros. Stellar aims to bridge the gap between cryptos and fiat money, making it easy to transfer money between them. In essence, this is a distributed global currency exchange.
