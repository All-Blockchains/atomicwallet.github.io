Binance Chain Fully Supported, AWC to Be Migrated

Recently, the world’s largest trading platform, Binance, has launched to its own decentralized blockchain called Binance Chain. The new blockchain’s features imply the BEP2 protocol powering fast transactions and higher capacity for DApps, in comparison to Ethereum. The native BNB coin will be moved to the new blockchain as well and fuel all in-chain processes.

We were highly anticipating this event. The forthcoming release of Atomic Wallet will fully support Binance Chain and BNB coin. Moreover, AWC token will be one of the first projects running on Binance Chain!

AWC on Binance Chain? How come?

AWC is set to be released on the base of the Binace blockchain. It is like an existing Ethereum-based token, but on Binance Chain. Both the AWC tokens are different and run on different blockchains. The AWC ticker, however, will stay the same.

What about ERC20?

We will keep the Ethereum-based version as well, so there will be the AWC token based on ERC20, and another AWC token based on Binance’s BEP2 protocol. To keep the token supply equal on each blockchain, we’ll burn 50 mln (50%) of ERC-20-based AWC and release that part on the Binance chain. You’ll be able to swap AWC between the chains.

How to swap AWC from ERC20 to Binance Chain?

For your convenience, we’ll arrange the swap via the Atomic Wallet apps and the website. The swap will be powered by one of our exchange partners, ChangeNOW, on a dedicated webpage. More information and detailed guides to come!
