Release 1.34.0

11/2/2018



Kindly check the largest new 1.34 release with plenty of enhancements and new features.

Main features:

  2. ZCash Sapling hardfork support
  3. Performance improved (check the details).
  4. Added USD balance and daily rates for every coin/token.
  5. UX improvements. We’ve added tips to help you manage XRP, XLM, VET and ETH properly.

New assets

  1. Added EOS-based tokens in Atomic Wallet.
  2. Zcash Sapling hardfork support.

Wallet

  1. Assets sorting by USD amount.
  2. Showing balance in USD for each coin and token (we are working on portfolio screen now).
  3. Showing a full name of assets and full ticker name.
  4. Added alert about insufficient ETH, VET, VTHO and for ERC20 token sending.
  5. Added alert about unspendable 20 XRP and 1 XLM deposit.
  6. Added 24h assets rate change.

History

  1. Enhanced loading time of transactions list and transactions details.

Exchange

  1. Coins sorting by the amount in the assets list.
  2. Blocking the “Exchange” button during the rate calculation.
  3. Added popup about funds sending to exchange.

Bug fixies

  1. Fixed ETH and ERC20 token sending error.
  2. Precise amounts when sending ETH, XML and TRX
  3. Fixed question mark (“?”) in currency names in Exchange.
  4. Exchange transaction status in History promptly updates.
  5. Empty exchange screen bug fixed.
  6. Empty exchange assets list bug fixed.
  7. Fixed app crashing when started and can’t connected to some blockchain.
  8. Fixed “Hide zero balance” button in Wallet.
  9. Increased DASH tx fee for properly transaction sending.
  10. Fixed bug with sending large TRX amount.
  11. Fixed bug with sending VET, XTZ.

Others

  1. Added alert before creating a new wallet with existing one.
  2. Re-designed the Exit popup window.

Release 0.1.33 beta

05/10/2018

We’ve implemented the option that allows you to buy cryptocurrencies for fiat and crafted the user interface by making it more comprehensive. We’ve also added Cardano (ADA) to complete the top-10 list of available cryptos. Below is the full list of all the updates you can enjoy in our freshly released 1.33 version.

New assets

  1. Cardano (ADA).

New features

  1. Buy crypto with your bank card directly in the Atomic Wallet interface. Assets availabe to purchase: (BTC, LTC, BCH, ETH). Supported cards: Visa and Mastercard.
  2. New pop-up hints prevent you from sending the incorrect amount of ETH, XRP or XLM.
  3. Now you can report errors and bugs directly to our support. Error descriptions will help us fix and resolve your issue way faster.
  4. EOS available for the instant exchange via Changelly.
  5. Nodes rotation for EOS. The wallet automatically switches to stable EOS nodes.

    6. Bugs fixed

    1. Sending the precise amount of all ERC20 tokens at once.
